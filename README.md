COVID-19 Data Analysis Project
Project Overview

This project performs a comprehensive analysis of COVID-19 time-series data for multiple countries. The analysis includes daily and weekly case calculations, rolling averages, peak detection, reproduction trend insights, and comparative visualizations.

The objective is to understand pandemic trends, identify outbreak peaks, and compare the progression of COVID-19 across different countries using data analytics techniques.

Project Objectives
Load country-wise COVID-19 time-series data.
Clean and preprocess the dataset.
Compute daily confirmed cases.
Calculate weekly case totals.
Generate rolling averages to smooth fluctuations.
Compare COVID-19 trends across countries.
Detect peak infection periods.
Estimate basic reproduction trend insights.
Visualize results using charts and graphs.
Export findings and conclusions.
